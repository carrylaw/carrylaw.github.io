想要分组添加序号，需要利用**@变量**，具体语句如下：
``` sql
select * from 
(select IF(@file=filename,@rank:=@rank+1,@rank:=1) as topic_rank,
        @file:=filename as filename,
        a.topic 
from transposition a) b;
-- 注意：@file变量需要先赋值
```
